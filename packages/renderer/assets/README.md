# Renderer Assets

This directory contains static assets used by the renderer process.

## Contents

- `icon.ico` - Windows application icon
- `icon.icns` - macOS application icon  
- `icon.png` - Linux application icon / fallback

## Usage

These icons are referenced by the main window manager (`packages/main/src/main/window.ts`) to set the window icon based on the platform.
